It’s not about memorizing system designs — it’s about understanding the fundamentals 
deeply enough to build something unfamiliar from scratch.

However, to get there, you need a strong foundation:

Learn the core concepts, terminology, and basic system patterns.
Study a few fundamental systems and understand how they work end-to-end.
Revisit and revise these until they become second nature.

The goal is to internalize these building blocks so well that, 
when faced with a new problem, you can confidently compose a solution from first principles.
